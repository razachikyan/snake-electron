<h1 align="center">Setup and Get Started</h1>

## Setup

- Clone the repository
  `git clone https://github.com/razachikyan/snake-electron.git`

- Open the project folder
  `cd snake-electron`

- Install the dependencies
  `npm install`

- Build the project
  `npm run build`

## Get Started

- Run the Game
  `npm start`
